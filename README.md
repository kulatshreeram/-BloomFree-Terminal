# 📊 BloomFree Terminal

A Bloomberg-inspired financial dashboard built using HTML, CSS, and JavaScript.

## Features

* 📈 Live Stock Market Data
* ₿ Cryptocurrency Tracking
* 💱 Forex Rates
* 📰 Financial News Feed
* 📊 Interactive Charts
* 💼 Portfolio Tracker
* 🇮🇳 India Market Dashboard
* 🌍 Multi-Timezone Clocks

## Tech Stack

* HTML
* CSS
* JavaScript
* Chart.js
* Finnhub API
* CoinGecko API
* Frankfurter API

## Run Locally

```bash
git clone <your-repo-link>
cd BloomFree-Terminal
```

Open `index.html` in your browser.

## Why This Project?

I wanted to create a free alternative inspired by Bloomberg Terminal to learn about financial data, APIs, dashboards, and real-time market visualization using only vanilla web technologies.
